# xy77.github.io

GitHub Pages editor shell and deployment pipeline for the `xy77/zen` project pages.

## Development

```bash
npm install
npm run dev
```

The Vite development server serves the editor at `http://localhost:5173/`.

## Production build

```bash
npm run build
PRIVATE_SOURCE_ROOT=/absolute/path/to/zen node .github/scripts/build-pages.js
```

Vite builds the public editor, authentication page, and preview page into `.build/app`.
The Pages builder then copies that application into `_site`, adds the generated project
pages and writes `projects.json`.

The GitHub Actions workflow checks out `xy77/zen` as `private-source`, runs both build
steps and deploys `_site` to GitHub Pages.

Previewed React content may load the Tailwind CDN independently. The editor shell itself
uses the CSS generated by Vite and Tailwind during the build.
